## Overview

Personal website.

### Development

Install dependencies:

```bash
pnpm install
```

Start a development server:

```bash
pnpm dev

# or start the server and open the app in a new browser tab
pnpm dev -- --open
```

### Building

To create a production version of the app:

```bash
pnpm build
```

To preview the production build:

```bash
pnpm preview
```
